# Шпаргалка: Вероятность выигрыша теннисного гейма

## 🎯 Суть задачи в одном предложении

**Найти вероятность выигрыша гейма Игроком 1, если он выигрывает каждое очко с вероятностью p.**

---

## 📊 Ключевые числа (для p = 0.6)

| Параметр | Значение |
|----------|----------|
| **Из состояния (40, 40)** | **69.23%** или **9/13** |
| Из состояния (A1, 40) | 87.69% (Advantage помогает!) |
| Из состояния (40, A2) | 41.54% (Advantage мешает!) |
| Из состояния (0, 0) | 73.57% (P1 имеет больше преимущества) |

## 🏗️ Архитектура программы

### Две стратегии решения

```
СПОСОБ 1: Матричный (точный)         СПОСОБ 2: Монте-Карло (приближенный)
────────────────────────────────────  ─────────────────────────────────
1. Построить матрицу P (20×20)       1. Симулировать N игр (N=1,000,000)
2. Разбить на блоки [Q R; 0 I]       2. Подсчитать побед P1
3. Решить (I-Q)u = R                 3. w ≈ (побед P1) / N
4. w = 0.692308                      4. ŵ ≈ 0.692268 ± 0.0009

Время: < 1 мс                        Время: ~ 2 сек
Точность: точно (до погрешности)    Точность: SE ≈ 0.0005
```

## 📐 Математика

### Формула для состояния (40, 40)

**Теоретическая формула:**
\[ w(p) = \frac{p^2}{p^2 + (1-p)^2} \]

**Для p = 0.6:**
\[ w(0.6) = \frac{0.36}{0.36 + 0.16} = \frac{0.36}{0.52} = \frac{9}{13} \approx 0.6923 \]

### Система уравнений для Deuce

```
w₀ = p·w₊ + (1-p)·w₋
w₊ = p·1 + (1-p)·w₀        [выигрыш или возврат]
w₋ = p·w₀ + (1-p)·0        [возврат или проигрыш]

Решение:
w₀ = p² / (p² + (1-p)²)
```

```

   (I - Q)u = R  →  u[15,0] = 0.692308 ✓

   692,268 / 1,000,000 = 0.692268 ✓ (отличие 0.006%)
```

## 🧠 Почему 69.23% > 60%?

**Интуиция:**
- P1 выигрывает каждое очко с вероятностью 60%
- Но из состояния "ровно" (40:40) нужно выиграть **на 2 очка больше**
- Это преимущество => вероятность > 60%

**Аналогия:**
```
Честная монета: P(орёл) = 50%
Но если нужна серия "два орла подряд" против "две решки подряд"?
P(два орла сначала) = 1/3... нет, это неправильно!

Правильная аналогия: "Кто первый выиграет в лучшем из 3?"
P(40:40) → нужны 2 выигрыша подряд
Это сложнее, чем просто выиграть одно очко.
```

## 🐛 Типичные ошибки при реализации

### Ошибка 1: Неправильная логика для (40, 40)

```python
# ❌ НЕПРАВИЛЬНО
if current_state == (40, 40) and player1_wins:
    return (50, 40)  # Не существует!

# ✓ ПРАВИЛЬНО
if current_state == (40, 40) and player1_wins:
    return ('A1', 40)  # Advantage для P1
```

### Ошибка 2: Неправильная логика для Advantage

```python
# ❌ НЕПРАВИЛЬНО
if s1 == 'A1' and player1_wins:
    current_state = ('A1', 40)  # Не меняется!

# ✓ ПРАВИЛЬНО
if s1 == 'A1' and player1_wins:
    return ('W1', 'L2')  # Победа!
```

### Ошибка 3: Забыли про возврат к (40, 40)

```python
# ❌ НЕПРАВИЛЬНО
if s2 == 'A2' and player1_wins:
    return (40, 40)  # Нужно именно это!
    # Но забыли в коде

# ✓ ПРАВИЛЬНО (в коде это есть)
elif s2 == 'A2':
    return (40, 40)
```
